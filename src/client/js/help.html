Paste you json and then provide a lodash based expression (Not limited to lodash only , any javascript expression can be used) , then click the Run button to evaluate the expression. The object jsonObj is the java object created from the entered json and we can play with this object in the expression.

Usage

  1. Run lodash filters and other functions to process json data and check the results without any code.
  2. Analyze test data for creating mocks, stubs etc.

Note: This website uses lodash 4.1.5 as is without any modification, we will move to higher versions as and when lodash releases one.

Slow ?

If the UI feels slow when processing huge JSON , try firefox which will definitely provide better performance. Meanwhile we are working on a server side processing so that the browser is not bothered with intensive data processing.....

Lodash

Lodash is an excellent utility providing a large set of capabilities to the javascript programming language. It is similar to LINQ in .net where javascsript objects can be passed to lodash functions to do sorting, fitering, map, search and many many functions. Learn more about lodash in its official documentation
